# Enterprise AI Knowledge Assistant

Enterprise-grade AI assistant for internal knowledge retrieval using **FastAPI, RAG, FAISS, Hugging Face, Streamlit, and Hexagonal Architecture**.

---

## Problem Statement

Employees often waste time searching for:

* Internal documents
* SOPs
* Policies
* API docs
* Process guides
* Old solutions

Knowledge is scattered across files, folders, emails, and shared drives.

---

## Solution

This project allows users to ask questions in natural language and get accurate answers with source references.

### Example Questions

* How do I reset my password?
* What is the leave policy?
* Explain the login API
* Summarize onboarding process
* Where is this information stored?

---

## Key Features

* Natural language Q&A
* Document search using RAG
* Local vector DB with FAISS
* Metadata-based source verification
* FastAPI backend
* Streamlit frontend
* Modular Hexagonal Architecture
* Free Hugging Face embeddings
